# Ning_Prompt (提示词工坊)

Ning_Prompt 是一个基于 Flet 开发的桌面端提示词（Prompt）优化与实验工具。旨在帮助用户通过 AI 的辅助，对原始提示词进行多维度的调整，以获得更符合预期的生成结果。

## ✨ 主要功能

本工具提供四种核心的提示词处理模式：

1.  **语义增强 (Enhance)**
    *   **作用**: 丰富原始提示词的细节。
    *   **效果**: 添加描述性形容词、风格修饰词、光影氛围描述，甚至补充推荐的参数（如 Midjourney 的 `--v` 或 `--ar`）。适用于从简单的想法生成高质量、细节丰富的图像或文本。

2.  **语义弱化 (Weaken)**
    *   **作用**: 概括和抽象化提示词。
    *   **效果**: 移除具体的细节指令，保留核心概念。适用于希望 AI 发挥更多自由想象力，或需要生成更具艺术感、概念性结果的场景。

3.  **语义修复 (Repair)**
    *   **作用**: 填补逻辑漏洞和消除歧义。
    *   **效果**: 修正语法错误，补充缺失的上下文，确保提示词的逻辑连贯性。适用于原始想法比较碎片化或表述不清的情况。

4.  **语义破坏 (Destroy)**
    *   **作用**: 碎片化重组，激发创意。
    *   **效果**: 打破原有的语义结构，提取关键词并进行非线性的重组。适用于寻找意外的灵感碰撞，或生成极具抽象风格的作品。

## 🚀 快速开始

### 1. 环境准备

确保您的系统已安装 Python 3.8+。

### 2. 安装依赖

在项目根目录下运行以下命令安装所需的 Python 库：

```bash
pip install -r requirements.txt
```

*(如果尚未创建 requirements.txt，请确保安装 `flet`, `httpx` 等核心库)*

### 3. 配置文件设置

项目默认不包含 `config.json` 文件。您需要从示例文件创建一个：

1.  找到项目根目录下的 `config.example.json` 文件。
2.  将其复制并重命名为 `config.json`。
3.  编辑 `config.json`，填入您的 API 信息：

```json
{
    "api_url": "https://api.deepseek.com/v1/chat/completions",
    "api_key": "YOUR_ACTUAL_API_KEY",
    "model": "deepseek-chat",
    "language": "zh",
    "theme_mode": "dark"
}
```

*   **api_url**: LLM 服务的 API 地址（支持 OpenAI 兼容接口，如 DeepSeek）。
*   **api_key**: 您的 API 密钥。
*   **model**: 使用的模型名称（例如 `deepseek-chat`, `gpt-4o` 等）。

### 4. 运行应用

```bash
python main.py
```

或者使用 Flet 命令运行：

```bash
flet run main.py
```

## ⚙️ 设置说明

在应用界面的右上角点击“设置”图标，可以进行以下配置：

*   **API 配置**:
    *   **API URL**: 接口地址。
    *   **API Key**: 鉴权密钥。
    *   **模型名称**: 指定调用的 AI 模型（如 `deepseek-chat`）。
*   **通用设置**:
    *   **语言**: 切换界面语言（支持中文/English）。
    *   **深色模式**: 切换应用的明暗主题。
